Standard Library
================

Math Functions
--------------

- **sqrt(x):** Returns the square root of x.

  .. code-block:: pepitoCode

     Float root = sqrt(16);  // root is 4.0

String Functions
----------------

- **length(s):** Returns the length of the string s.

  .. code-block:: pepitoCode

     Int len = length("Hello");  // len is 5
