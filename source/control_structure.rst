Control Structures
==================

If-Else Statement
-----------------

.. code-block:: pepitoCode

   if (condition) {
       // code to execute if condition is true
   } else {
       // code to execute if condition is false
   }

For Loop
--------

.. code-block:: pepitoCode

   for (Int i = 0; i < 10; i++) {
       // code to execute for each iteration
   }

While Loop
----------

.. code-block:: pepitoCode

   while (condition) {
       // code to execute while condition is true
   }
